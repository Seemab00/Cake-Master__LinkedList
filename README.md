# Cake Master Management System

## 📌 About

**Cake Master Management System** is a web-based platform developed for the Data Structures and Algorithm course. It simplifies cake ordering, customization, and management for both customers and admins through a digital solution.

---

## ✨ Features

### Customer
- Sign up & login
- Browse cake catalog (descriptions, sizes, prices)
- Customize & place orders
- Track order & delivery status
- Payment options (EasyPaisa, Cash on Delivery)
- Feedback & ratings

### Admin
- Manage customer accounts (view, update, delete)
- Catalog management (add, update, remove cakes)
- Order management (track, update status, cancel)
- Financial reporting (sales reports)
- Discount control
- Delivery control (range & options)

---

## 🛠️ Tech Stack

- **Language**: Java
- **Data Structure**: Linked List
- **Methodology**: Agile
- **Interface**: Web-based
